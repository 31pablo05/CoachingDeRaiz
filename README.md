# 🌱 Coaching de Raíz — Lucía Vallejo

Sitio web profesional de **Lucía Vallejo**, Coach Ontológica Empresarial, Ingeniera Civil y Project Manager.

**Sitio en producción:** https://www.coachingderaiz.com

---

## 🛠️ Stack

- **Astro 5** — SSG (Static Site Generation)
- **React 18** — Componentes interactivos
- **Tailwind CSS 3** — Estilos
- **Vercel** — Hosting y deploy automático

---

## 📁 Estructura

```n src/
 ├── layouts/
 │   └── BaseLayout.astro      ← Layout con SEO completo
 ├── pages/
 │   ├── index.astro           ← Página principal (single page)
 │   ├── servicios.astro       ← Redirect a /#servicios
 │   ├── sobre-mi.astro        ← Redirect a /#sobre-mi
 │   ├── que-es-coaching.astro ← Redirect a /#coaching
 │   ├── contacto.astro        ← Redirect a /#contacto
 │   ├── blog/index.astro      ← Blog (próximamente)
 │   └── 404.astro             ← Página de error
 ├── components/               ← Componentes React (.jsx)
 └── styles/
     └── global.css
```n
---

## 🚀 Comandos

```bash
npm install       # Instalar dependencias
npm run dev       # Servidor local (puerto 4321)
npm run build     # Build de producción
npm run preview   # Preview del build
```n
